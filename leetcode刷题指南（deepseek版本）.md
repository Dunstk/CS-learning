# 算法提升路径
## 🧱 一、打牢基础：数据结构 + 算法原理

- 学习常见数据结构：数组、链表、栈、队列、哈希表、树、图、堆等
    
- 掌握基础算法：排序、查找、递归、分治、贪心、动态规划、回溯、图遍历（DFS/BFS）
    
- 推荐书籍：
    
    - 《算法图解》：适合入门，图文并茂
        
    - 《数据结构与算法分析》：适合系统学习
        
    - 《算法（第四版）》：偏理论，适合进阶
        

## 🧪 二、持续练习：刷题 + 竞赛

- **刷题平台**：
    
    - LeetCode：题目分类清晰，适合系统训练
        
    - Codeforces：偏竞赛风格，适合挑战
        
    - 牛客网：适合国内比赛和校招准备
        
- **刷题建议**：
    
    - 按专题刷题，如“二分查找专题”“动态规划专题”
        
    - 每种算法刷 30~50 道题，形成“题感”
        
    - 做完题后**总结思路、记录错因、复盘优化**
        

## 🧠 三、提升思维：归纳总结 + 模板积累

- 建立自己的“算法笔记本”，记录：
    
    - 常见题型的解法套路
        
    - 易错点和优化技巧
        
    - 模板代码（如并查集、KMP、Trie）
        
- 每周进行一次复盘：回顾本周做过的题，归类总结
    

## 🏆 四、参与竞赛：实战锻炼 +团队协作

- 适合新生的比赛：
    
    - 天梯赛、CSP-J/S、校内程序设计赛
        
- 竞赛能锻炼：
    
    - 编程速度
        
    - 多题并行思考能力
        
    - 团队协作与交流
        

## 📅 五、制定计划：阶段性目标 + 日常任务
